Instruction rules
  • Always replace "Z" with a value or defined constant.
  • Only write the names of other registers with these instructions:
    • REGSET [register name] [register name | constant name | value]
    • ACSET [register name | value]
  • These instructions are special:
    • 0x6x range: 0x6x instructions need a value. This instruction does not need or let you type the AC name.
    • 0x7x range: 0x7x instructions need the MEM name. This instruction does not need or let you type the AC name.
  • Never type commas in the instructions. The only separator in an instruction is the space.
